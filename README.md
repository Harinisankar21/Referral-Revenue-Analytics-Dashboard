# 📊 Referral Revenue Analytics Dashboard

## Overview

This dashboard was developed during my internship at **Niftrix Technologies** using Microsoft Power BI.

It provides insights into the company's referral program, including referral performance, revenue generation, premium membership statistics, and monthly trends.

> Note: The original Power BI (.pbix) file is not included because it contains company data.

---

## Dashboard Features

- Total Users
- Referrals Given
- Referrals Received
- Total Revenue
- Premium Members
- Regular Members
- Monthly Revenue Analysis
- Referral Type Analysis
- Revenue by Referral Type
- User-wise Referral Analysis

---

## Tools Used

- Microsoft Power BI
- DAX
- Data Visualization
- Data Modeling
